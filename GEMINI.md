In diesem Projekt wird die WebSeite von logismy.de erstellt. Dies ist eine Schwesterseite von kategory.de. kategoroy.de
muss zu diesem Zweck noch überarbeitet werden. Aber das ist hier nicht Thema.

In allen Dateien mit Einrückung, rücken wir immer 3 Spaces, keine TAB, ein.

Wir nutzen Bootstrap 5, Es soll so verwendet werden, dass "Responsive Design" optimal unterstützt wird.

Es handelt sich hier um eine WebSeite, die auf GitHub gehostet werden können muss.

Grundlegende css Einstellungen sollen in einer kategory.css im root abgelegt werden.

kategory ist light-mode und logismy im dark-mode. Das sollte fast die einzige Einstellung in der logismy.css sein.

Es gibt ein Logo in der Datei kategory.svg. Das soll auf beiden sites eingebunden werden.

Die Schritarten sind:
- font-family: "Times New Roman", Times, Baskerville, Georgia, serif;
- Für Mono-Space nutzen wir den Font-Stack: font-family: 'Consolas', 'Monaco', 'Menlo', 'Ubuntu Mono', 'source-code-pro', monospace;

Es folgen die Farben in RGB. Sie sind im Dark - und Light-Mode in gleicher weise zu verwenden.

Dunkelgrau: 87/86/86
Rot       : 227/6/19

Grün      : 0/234/0
Gelb      : 252/228/0
Orange    : 255/126/0
Hellblau  : 0/240/240
Dunkelblau: 25/118/225

Weiß      : 255/255/255
Schwarz   : 0/0/0

Die normalen Farben für Text und so sind Schwarz und Weiß.

Rot nutzen wir für sehr wichtige Akzente
Dunkelgrau für geometrische gestalltungen (Rahmen, dunkle Flächen ... )

kategory und logismy sollen zunächst nur eine Startseite haben: index.html
- Bei kategory das logo in groß in der Mitte
- Bei logismy das logo oben links in kleiner. Auf der Mitte der Seite ein sehr großes \lambda (der griechische Buchstabe, das kleine Lambda)

kategory wird spätetr über kategory.de erreicht werden
logismy soll später über logismy.de erreicht werden
