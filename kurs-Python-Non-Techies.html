Was ist überhaupt ein Programm?

Das wolltest Du schon immer wissen, aber die “Anfänger”-Kurse legen immer richtig los. Die anderen können immer schon so viel. Du hast hingegen noch nie eine Zeile Code gesehen, geschweige denn getippt.

Dann bist Du hier goldrichtig: von Variablen, Schleifen und Funktionen bis hin zum KI-Agenten-Einsatz (Vibe-Coding).

Editor, Code, IDE, Variable, Funktion, Datenbank, Module, Interface, Klasse, Unit-Test, Parsen, Frontend/Backend, Deployen: wir lernen ein wenig IT-Latein.
